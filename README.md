## Project Description

Analyze electricity generation and consumption patterns in Ireland using open data. Explore trends, visualize outputs, and learn data engineering and analysis skills.

## Dataset

- **Name:** MEG01 - Metered Electricity Generation  
- **Source:** [Ireland Open Data](https://data.gov.ie/dataset/meg01-metered-electricity-generation?utm_source=chatgpt.com)  
- **Size:** ~100,000 records  
- **Format:** CSV / JSON / XLSX
